Java Web的一次实践

发表于 2018-09-22 分类于 Java 本文字数： 744
Web项目.

本次项目为JSP+Servlet+Java Bean+DAO层实现的传输数据到服务器、存储进数据库数据的一个项目。

JSP

JSP主要用来做页面，JSP本是Servlet的扩展，但如今已是纯HTML的了，但是与HTML的不同在于：可以嵌入Java代码、jstl标签语言（不过这么干不太规范…）

  • 写页面
    • html语言（加上css样式，这个是真费时间啊，毕竟自己审美排版啥的就。。）
    • form表单以post方法传输给servlet（服务器层吧），普通的就getParameter()获取，难点就在于获取上传图片的二进制流，之前也用getInputStream()得到，然后转换成bytes，最后转成String,获取其中的书名、作者、简介…但这个太麻烦了，后来用ServletFileUpload和FileItem迭代得到所有信息，同时将图片写入服务器端文件夹下
    • JS传数据给servlet：form表单、url后跟参数、
    • 用js限制输入格式，包括不能为空，不能超过输入字数…
    • 用jstl语言轮询数据库，显示符合条件的数据

Servlet

  • 接收数据
    • 就是用来接收数据的，从JSP接收，再用Java Bean封存
    • Java Bean是一个特殊的类，用来封存数据库表的属性的类，里面有一些访问属性的getter和设置属性的setter
    • 再用DAO层进行底层操作（数据库的增删查改）
  • 传数据
    • 返回信息到下一页面（跳转jsp页面），有时需要传数据，url后跟参数，session.setAttribute(),request.setAttribute(),（二者之间持续时间不同。。）
    • 难点在于跳转jsp需要接收上一个JSP和Servlet的数据，在这个JSP里必须用同样的表达式才能在跳转两次后页面不变化
