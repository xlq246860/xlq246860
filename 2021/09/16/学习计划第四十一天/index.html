不同的二叉搜索树

思路

遍历[1, n]区间内的数，以当前数为根节点，计算其左、右子树的个数，对子树的计算采用递归，结束条件是区间不存在或只有一个值时，返回1；

伪代码

buildTree(int left, int right)
{
if(left>=right)
{
return 1;
}
int count=0;
for(int i=left;i<=right;i++)
{
// 以i为根节点
int l=buildTree(left, i-1);
int r=buildTree(i+1, right);
count+=(l*r);
}
return count;
}

复杂度

时间复杂度：O(nlogn)

连续子数组的最大乘积

思路

想采用动规，但是由于存在负数，dp[i]表示以i结尾的最大乘积不能得到全局最大，因为有可能存在负数，使得结果小，之后再遇到负数，使结果最大。所以采用两个动规数组，一个记录最大值，一个记录最小值，每个数的最大值取决于前一个数结尾的最大值和最小值与当前数的乘积结果，最小值也取决于前一个数的最大值和最小值与当前数的乘积。
