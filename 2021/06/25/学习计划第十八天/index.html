方格取数.

方格取数

思路

采用动规的方式进行解决。

  1. 开始的时候考虑的是用动规做两遍，dp[i][j]的思路，但这是贪心的做法，并不能得到最优解。
    如：
    0 3 5
    2 6 0
    0 4 0
    最优解是：0->2->6->4->0; 0->3->5->0->0.
    但按照上面的动规方法产生的结果为：0->3->6->4->0; 0->3(去掉)->5->0->0.
  2. 动规，四维的。dp[i][j][k][l]：两个同时所在的位置产生的和最大。
    dp[i][j][k][l]=max(dp[i-1][j][k-1][l], dp[i][j-1][k-1][l], dp[i-1][j][k][l-1], dp[i][j-1][k][l-1])+map[i][j]+map[k][l].
    if k=i&&l=j; then dp[i][j][k][l]-=map[k][l];
  3. 三维动规。dp[i][j][l]，走了l步，第一条路(i, l-i)，第二条路(j, l-j)，得到的两个法力的之和最大。dp[i][j][l]=map[i][l-i]+map[j][l-j]+max(dp[i][j][l-1], dp[i-1][j][l-1], dp[i-1][j-1][l-1], dp[i][j-1][l-1]).

复杂度

时间复杂度：O(n^4) -> O(n^3)
空间复杂度：O(n^4) -> O(n^3) -> O(n^2)
