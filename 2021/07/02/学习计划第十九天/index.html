统计值等于子数平均值的节点数

思路

利用后序遍历的方式，得到每个节点的左子树和右子树的节点数和节点值的和，然后计算该节点值是否等于子树平均值。

空间复杂度：O(n)。递归栈最坏可能会O(n)，每个节点一层。

统计打字方案数

思路

一开始是采用递归的方式来模拟组合字符。
递归的方式都可以采用动规的方案来做。
（ ↓ 这是针对同一个字符所做的动规转移方程）
dp[i]=dp[i-1]+dp[i-2]+dp[i-3] if c!=’7’&&c!=’9’
dp[i]=dp[i-1]+dp[i-2]+dp[i-3]+dp[i-4] if c!=’7’||c!=’9’
但是需要注意乘积太大，导致的溢出。结果不正确。
所以动规应该针对所有字符来操作。
dp[i]=dp[i-1]; s[i]!=s[i-1]
dp[i]=dp[i-1]+dp[i-2]; s[i]=s[i-1]
dp[i]==dp[i-1]+dp[i-2]+dp[i-3];
dp[i-1]+dp[i-2]+dp[i-3]+dp[i-4]; s[i]=s[i-1]=s[i-2]=s[i-3].
if i-1, i-2, i-3, i-4不存在的话，则改为+1.

复杂度

时间复杂度：dp为O(n)
空间复杂度：dp为O(n)

检查是否有合法括号字符串路

思路

采用递归的方法模拟在网格图中搜索，利用栈进行括号的压入和弹出。利用剪枝避免重复和不必要的搜索。

  • 当在位置(i, j)时，若栈内括号数大于未走的路径长度，即使后面全部为’)’也不能使括号合法。
  • 当栈的size=0时，若遇到’)’，返回false。
  • 利用visit[i][j][c]记录已经访问过的情况。(i, j)为位置，c是在该位置时，栈内括号数，若已访问过，则返回false。因为若是访问过且合法已经返回true不会再去dfs。

可以采用变量c来记录括号数，c>=0才是合法的，当遇到左括号+1，遇到右括号-1。

复杂度

时间复杂度：O(mn(m+n))，所有状态。 // 不太懂
空间复杂度：主要是vis数组，O(mn(m+n))
