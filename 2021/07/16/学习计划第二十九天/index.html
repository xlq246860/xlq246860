组合；简化文件路径；不同路径

组合

思路

排列组合，从n个数中选出k个。
采用的是递归的方式，对于每个数进行二进制枚举，可以选择加入该数或者不加入。

复杂度

时间复杂度：O(2^n)
空间复杂度：O(n)，递归栈。

简化文件路径

思路

因为遇到”..”需要变成上级目录，利用栈弹出当前目录，遇到”.”则什么也不做，遇到其余目录名则压入栈内。以”/ /“拆分目录名。

伪代码

复杂度

时间复杂度：O(n)
空间复杂度：O(n)

不同路径

思路

可以用递归的方式解决的就能用动规解决。
dp[i][j]=dp[i-1][j]+dp[i][j-1]; // 到达位置(i, j)的路径数为从位置(i-1, j)和(i, j-1)的路径数之和。
dp[i][j]=0; // if grid[i][j]==1，有障碍物
