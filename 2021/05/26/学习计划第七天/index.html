鸡蛋掉落

思路

这题比较难，还没理解动规方程的含义。
解法一：逆向思维。dp[t][k]表示在t个操作k个鸡蛋最高能到达的楼层数。


dp[i][j]=dp[i-1][j-1]+dp[i-1][k]+1;
dp[1][j]=1;
dp[i][1]=i;
for(int i=0;i<=n;i++)
{
if(dp[i][k]>=n)
{
return i;
}
}

复杂度

时间复杂度：O(2^n) ?
空间复杂度：递归栈 O(n)
